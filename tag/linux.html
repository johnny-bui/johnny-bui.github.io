Configure Okular to search backward a TeX file from a PDF file

Tue 22 December 2015 | in Others | tags: linuxkdetexvim

Short Version

Setting -> Configure Okular -> Editor ->

Editor: Custom Text Editor Command: There are some configurations for diverse not listed editor.

  • For gvim: gvim --remote-tab-silent +%l %f
  • For texmaker: texmaker -use -line %l -column %c

Long Version

Latex supports reverse search. That is, once you compile your Latex Document with the option -synctex=1, the Latex compiler will write a file name *.synctex.gz. Some PDF Viewer can read this file when it open a PDF File, and open a editor, lets the cursor of the Editor jump to a position of the tex file. One of these PDF Viewer is Okular.

With the configuration in the Short Version you can use Okular as your PDF Viewer, once you click on the document with holding shift, Okular will open your editor (here gVim or kate) with the tex file, let the cursor of the editor jump to the position, which matches the clicked position of the PDF file.

So Happy texing!.
